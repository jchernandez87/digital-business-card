### Prerequisites

* Node
* React


### Installation

1. Clone the repo
   ```sh
   git clone git@github.com:jchernandez87/digital-business-card/
   ```
2. Install node packages
   ```sh
   npm install
   ```
3. Run the server
   ```sh
   npm start
   ```
